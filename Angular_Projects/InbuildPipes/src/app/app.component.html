Demonstration of inbuild pipes

1. Date pipes

Transforms a date value into a formatted string.

{{today}}

Today is :{{today | date:'fullDate'}}

Today is :{{today | date:"short"}}

Today is :{{today | date:"shortDate"}}

Today is :{{today | date:"shortTime"}}

2. String pipes

Transforms a date value into a formatted string.

{{message}}

message is :{{message | uppercase}}

message is :{{message | lowercase}}

message is :{{message | titlecase}}

message is :{{message | slice:8}}

3. JSON pipe

{{data | json}}

4. Number pipe

number is :{{no | number : '1.2-3'}}

number is :{{no | number : '3.2-5'}}

number is :{{no | number : '3.4-5'}}

5. Pipe for currency

{{rate | percent}}

{{78 | currency}}

{{78 | currency: 'INR'}}
